1. Download the PHPMailer from GitHub (https://github.com/PHPMailer/PHPMailer), Extract, copy the PHPMailer and Paste to Project Directory.
2. Write config.php and send_mail.php
3. Ensure the 2-factor authentication of gmail and APP Password. Remove the unnecessary spaces from the APP Password and copy to config file instead real password of your gmail.
4. run the send_mail.php
